[npm][]:

```sh
npm install hebo-validation
```

[yarn][]:

```sh
yarn add hebo-validation
```


## Usage

```js
const HeboValidation = require('hebo-validation');

const heboValidation = new HeboValidation();

console.log(heboValidation.renderName());
// script
```
