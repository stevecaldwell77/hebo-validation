## Install

[npm][]:

```sh
npm install hebo-validators
```

[yarn][]:

```sh
yarn add hebo-validators
```


## Usage

```js
const HeboValidators = require('hebo-validators');

const heboValidators = new HeboValidators();

console.log(heboValidators.renderName());
// script
```
